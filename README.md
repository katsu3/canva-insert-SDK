# canva-insert-SDK

## セットアップ

### ローカルサーバーの起動

```bash
python3 server.py
```

サーバーが起動したら、ブラウザで以下にアクセス:
```
http://localhost:8080
```

サーバーが起動したら、ブラウザで以下にアクセス:
```
http://localhost:8000
```

## 使い方

1. ローカルサーバーを起動
2. `http://localhost:8080` にアクセス
3. Google DriveのフォルダIDを入力
4. 「画像一覧を読み込む」をクリック
5. 「すべての画像を自動配置」でCanvaに画像を挿入

## キャッシュ問題の解決方法

Canvaのプレビューでコード変更が反映されない場合：

1. `version.json` のバージョン番号を変更
   ```json
   {
     "version": "1"
   }
   ```
   ↓
   ```json
   {
     "version": "2"
   }
   ```

2. Canvaプレビューをリロード

**理由:** Canvaのiframeが強くスクリプトをキャッシュするため、バージョン番号の変更で確実に最新版を読み込ませる必要があります。
